#  Skin Disease Detection System using CNN

A deep learning-based skin disease detection system that uses Convolutional Neural Networks (CNN) to classify skin conditions such as:

- Eczema
- Nail Fungus & Other Nail Diseases
- Malignant Lesions

This project was developed as a final year project in my Bachelor's degree in Computer Science. The backend model was trained using Google Colab, and an API was developed using Python (Flask/FastAPI) in VS Code. The project features a simple front-end application for user interaction.

---

## 🚀 Features

- ✅ Image-based disease detection using CNN
- ✅ Trained on a publicly available dataset
- ✅ Frontend interface for login, diagnosis, feedback, etc.
- ✅ REST API integration
- ✅ Responsive and user-friendly design
